Debugging Information

Open Chrome DevTools (F12) and check the Console tab for extension logs.

If the extension isn't working:

  • Make sure the extension is loaded in chrome://extensions/
  • Check for errors in the background page console
  • Check for errors in the content script console
  • Try reloading the extension
