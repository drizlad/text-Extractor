# Extension Icons

This folder contains the icon files for the Chrome extension:

- `icon-16.png` - 16x16 pixels (toolbar icon)
- `icon-48.png` - 48x48 pixels (extension management page)
- `icon-128.png` - 128x128 pixels (Chrome Web Store)

## Current Status

**PLACEHOLDER ICONS**: The current icons are minimal PNG placeholders. **You must replace these with proper icons before Chrome Web Store submission.**

## Icon Requirements

### Technical Requirements
- **Format**: PNG with transparency
- **Sizes**: Exactly 16×16, 48×48, and 128×128 pixels
- **File Size**: Keep under 100KB per icon
- **Transparency**: Use transparent backgrounds
- **Square**: All icons must be square

### Design Guidelines
- **Simple Design**: Icons should be clear and recognizable at small sizes (especially 16×16)
- **Text Extraction Theme**: Should suggest text, selection, or extraction functionality
- **Chrome Theme**: Use blue colors (#1a73e8, #4285f4) to match Chrome's design
- **High Contrast**: Ensure good visibility on various backgrounds

### Suggested Icon Concepts
- Document with text selection highlight
- Magnifying glass over text
- Cursor selecting text
- Text bubble with extraction arrow
- Camera/text combination

## Creating Icons

### Tools
- **Free**: GIMP, Inkscape, Canva
- **Professional**: Adobe Illustrator, Sketch, Figma
- **Online**: Icon generators, favicon creators

### Process
1. Design the icon at 128×128 pixels
2. Scale down to 48×48 and 16×16
3. Optimize for each size (simplify details for smaller sizes)
4. Export as PNG with transparency
5. Test visibility at all sizes

## Chrome Web Store Submission

Before submitting to Chrome Web Store:

1. **Replace all placeholder icons** with professional icons
2. **Test icons** in Chrome extension management page
3. **Verify transparency** works correctly
4. **Check file sizes** are reasonable
5. **Test on different backgrounds** (light/dark themes)
